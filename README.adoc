= b2-cookiecutter

____
Cookiecutter template for C++ projects using Boost.Build.
____

== Install

Cookiecutter can automatically pull templates from GitHub, see <<Usage>>.

== Usage

[source,shell]
----
$ cookiecutter gh:grisumbras/b2-cookiecutter
----
